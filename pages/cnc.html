Link do uputstva kako se inicijalizuje arduino za cnc: https://www.youtube.com/watch?v=oIKEYM-lMWQ&t=895s

Skinuti grbl i kopirati u libraries folder uputstvo git clone https://github.com/gnea/grbl.git cd grbl cp -r grbl ~/Arduino/libraries Naci grblUpload iz File > Examples > grbl > grblUpload

Aplikacija za slanje G kodova je Universal G Code sender java Aplikacija i skida se sa github i pokrece se iz komandne linije ``` ~/Programs/ugsplatform-linux/bin/ugsplatform ``` Treba podesiti konekciju: Firmware GRBL, port /dev/ttyACM0 i baud rate 115200. Proci kroz `Machine > Setup wizard` da bi se podesio korak na 83. Meni je isao na 10.56 umesto na 10, tako da sam sa 83 smanjio na 79.75. Treba da se klikne na Update i posle toga vizard se moze cancelovati. Omoguciti Pendant Machine > Pendant pa pristupiti preko telefona na lokalnoj mrezi http://192.168.1.190:8080/ 'Reset zero' podesava poziciju na 0, bez pomeranja uredjaja. Postoje dva broja, jedan je pozicija od nule tj zadnjeg reset zero (work), a druga je pozicija od ukljucivanja programa (machine). Kada je u alarmu moze se izaci sa 'Kill alarm' ili 'Soft reset'
